Мини-приложение для просмотра прогноза погоды.

## Запуск приложения

1. Установите все необходимые зависимости

```bash
npm i
```

2. Запустите сервер:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

---

По умолчанию сервер будет расположен на [http://localhost:5173/](http://localhost:5173/).
